# WebGarcH
Web Game Archive  (With Emulators and at some point Flash)

# How To Start & Install (Req. Python 3, Git)
How To Install:
```
git clone https://github.com/QuartzWork/WebGarcH
cd WebGarcH
python(3) compiler.py 
```
How To Start:
```
git clone https://github.com/QuartzWork/WebGarcH
cd WebGarcH
python(3) -m http.server *PORT*
```
